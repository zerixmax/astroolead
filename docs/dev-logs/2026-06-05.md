* **Popravak Slideshow Lightboxa (fslightbox):**
  - **CSS overrides za z-index i SVG ikone:** Dodani su specifični stilovi u [global.css](file:///home/z3r1x/Dokumenti/astroolead/src/styles/global.css) koji rješavaju CSS konflikt (stacking context).
